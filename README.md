## üìñ About

Dragon's Escape is a terminal-based adventure game that combines classic text adventure gameplay with modern Rust programming. Explore a mysterious dragon's castle, navigate through beautifully rendered ASCII art rooms, and uncover hidden treasures in this immersive command-line experience.

## üöÄ Quick Start

### Prerequisites
- [Rust and Cargo](https://rustup.rs/) installed on your system

### Installation & Play
```bash
# Clone the repository
git clone https://github.com/NgunjiriBrenda/Text-Based-Adventure-Plan.git

# Navigate to project directory
cd Text-Based-Adventure-Plan

# Run the game
cargo run

# Debug build (for development)
cargo build

# Release build (for distribution)
cargo build --release

# Run tests
cargo test

# Check code quality
cargo clippy

# Format code
cargo fmt
Dependencies
Standard Rust libraries only

No external crates required
